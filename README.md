####
- Parametrização da porta do container

```{{ .Values.service.targetPort }}````

```
service:
  type: ClusterIP
  port: 80
  targetPort: 80
````

- Parametrização do liveness e readiness probes

```yaml
          {{- with .Values.livenessProbe }}
          livenessProbe:
            {{- toYaml . | nindent 12 }}
          {{- end }}
          {{- with .Values.readinessProbe }}
          readinessProbe:
            {{- toYaml . | nindent 12 }}
          {{- end }}
```
